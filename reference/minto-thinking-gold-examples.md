각 예시는 프롬프트에 포함될 때 **실제 출력과 동일한 형식**으로 작성되었다.

## 예시 1: 단순 정방향 구조화 (Simple Forward)

**입력 텍스트**:
```
우리 팀은 최근에 생산성이 많이 떨어지고 있다. 
회의가 너무 많아서 실제 작업 시간이 부족하고, 
우선순위가 명확하지 않아서 중요한 일을 자꾸 미루게 된다. 
팀원들도 번아웃 증상을 호소하고 있다. 어떻게 해야 할까?
```

**주요 판단 포인트**:
- CS: "현재 문제"와 "근본 원인"으로 명확히 분리
- GPS: 문제 현상은 contain, 원인은 branch
- Pyramid: Level 1을 3개로 구성 (회의 과다, 우선순위 부재, 번아웃 악순환)
- MECE: 상위 3개는 상호 배제되면서 전체를 포괄

**생성된 사용자 출력 (Markdown)**:

```markdown
---
tags: [minto, think]
created: 2026-06-18
type: thought
coreClaim: "회의 과다와 우선순위 불명확성이 팀 생산성 저하와 번아웃의 핵심 원인이므로, 주간 집중 시간 확보와 명확한 우선순위 체계 도입이 가장 시급하다."
---

## CS 분석 (범주·요점)

### 범주 1: 현재 문제 현상
- 요점: 생산성 저하, 중요한 작업 지연, 팀원 번아웃 증상 호소

### 범주 2: 주요 원인
- 요점: 과도한 회의, 우선순위 불명확, 작업 시간 부족

> 추론 보충 범주: "번아웃 악순환" — 생산성 저하가 다시 회의 증가와 동기 저하를 유발하는 피드백 루프가 존재함을 보충

## GPS 구조화

### 그룹 구조

```
팀 생산성 문제
├── 문제 현상 (Contain)
│   ├── 생산성 저하
│   ├── 중요한 일 미루기
│   └── 번아웃 증상
└── 근본 원인 (Branch)
    ├── 회의 과다
    ├── 우선순위 불명확
    └── 작업 시간 부족
```

### 비교 분석 (패러렐)
| 항목          | 영향                  | 즉시 대응 방안          |
|---------------|-----------------------|-------------------------|
| 회의 과다     | 실제 작업 시간 감소   | 주간 집중 시간 블록 확보 |
| 우선순위 불명확 | 중요한 일 지연       | 주간 우선순위 명시 회의 |
| 번아웃        | 생산성 추가 저하      | 회의 축소 + 회복 시간 확보 |

### 순서 분석 (시리즈)
```
회의 과다 → 작업 시간 부족 → 우선순위 혼란 → 중요한 일 미룸 → 번아웃 악화
```

## 핵심 주장

> 회의 과다와 우선순위 불명확성이 팀 생산성 저하와 번아웃의 핵심 원인이므로, 주간 집중 시간 확보와 명확한 우선순위 체계 도입이 가장 시급하다.

## 피라미드 구조

### 근거 1: 과도한 회의가 실제 작업 시간을 심각하게 침해하고 있다
- 하루 평균 회의 시간이 3시간 이상을 차지
- 집중이 필요한 작업이 회의 사이에 쪼개져 효율이 떨어짐
- "회의가 끝나면 이미 피곤하다"는 팀원 피드백 다수

### 근거 2: 우선순위가 명확하지 않아 중요한 일이 지속적으로 밀리고 있다
- 어떤 일이 진짜 중요한지 팀 전체가 공유하지 못함
- 긴급한 일에만 대응하는 문화가 고착화됨
- 분기 목표와 일일 작업의 연결고리가 약함

### 근거 3: 생산성 저하와 번아웃이 서로를 강화하는 악순환을 만들고 있다
- 작업이 밀릴수록 야근과 스트레스 증가
- 번아웃으로 집중력이 떨어져 더 많은 회의가 필요해지는 구조
- 이 악순환을 끊지 않으면 단기 개선으로는 한계가 있음

## MECE 검증

- 결과: **통과**
- 로직트리 규칙: 엄격 1~2단계 적용 / 3단계 이후 완화
- 검토 사항:
  - 근거 1(회의), 근거 2(우선순위), 근거 3(번아웃 악순환)은 명확히 구분됨
  - 세 가지가 상호 배제되면서 현재 관찰되는 모든 주요 증상을 포괄

## 요약

팀 생산성 저하의 주된 원인은 과도한 회의와 불명확한 우선순위다. 이 두 가지가 작업 시간을 잠식하고 의사결정 혼란을 일으키며, 결국 번아웃 악순환까지 연결된다. 주간 집중 시간 확보와 명확한 우선순위 체계를 동시에 도입하는 것이 가장 효과적인 출발점이다.
```

## 예시 2: 복잡 정방향 구조화 (Complex Forward — Parallels + Series)

**입력 텍스트**:
```
반도체 슈퍼사이클이 아직 유효한 상황에서, 금리 상승과 지정학 리스크 때문에 
한국 주식 시장이 불안하다. BOJ 금리 인상, 엔캐리 청산 우려, 
미국 중간선거 불확실성 등이 겹쳐 있다. 
그럼에도 AI 수요는 계속 증가하고 있고, 삼성전자와 SK하이닉스가 
코스피 이익의 70%를 차지하고 있다. 
이 상황에서 포트폴리오 전략을 어떻게 세워야 할까?
```

**주요 판단 포인트**:
- CS: 6개 범주로 세밀하게 추출 (시장 노이즈, 금리 충격, 사이클 지속성, 재평가 조건, 섹터 확산, 전략 결론)
- GPS: Branch 구조 + Parallels (금리 이슈별 충격 비교표) + Series (의사결정 순서)
- Pyramid: 4개 Level 1 (금리 노이즈, 사이클 지속, 멀티플 재평가, 섹터 확산)
- typeRationale을 구체적으로 작성

**생성된 사용자 출력 (Markdown)**:

```markdown
---
tags: [minto, think]
created: 2026-06-18
type: thought
coreClaim: "금리 노이즈와 지정학 불확실성은 단기 충격에 그치므로, 반도체 슈퍼사이클이 유효한 지금은 삼성전자·SK하이닉스 중심 50% 이상 비중을 유지하되 멀티플 재평가 조건과 섹터 순환매를 병행하는 전략이 가장 현실적이다."
---

## CS 분석 (범주·요점)

### 범주 1: 시장 노이즈와 파국 회피
- 요점: 종전 합의는 '잠시 멈춤' 수준, 미국도 파국을 스스로 피할 유인이 강함

### 범주 2: 금리 이슈의 실제 충격
- 요점: BOJ 금리 1%대 진입은 예상 범위, 엔캐리 청산 우려 약화, FOMC 동결 기대

### 범주 3: 반도체 슈퍼사이클 지속성
- 요점: AI 하이퍼스케일러 투자 지속, 고점 신호(유증/IPO) 아직 미관찰, 삼성·하이닉스 중심 지위 불변

### 범주 4: 반도체 주가 재평가 조건
- 요점: 이익 성장 기울기 완만화 → 멀티플 재평가 필요, M&A·주주환원 강화 필요

### 범주 5: 섹터 확산과 순환매
- 요점: 반도체 공백기에만 타 섹터 확산 유효, 기판·장비·소재 순환매 형태

### 범주 6: 투자 전략 결론
- 요점: 반도체 50%+ 유지 + 순환매 + 변동성 시 축소 후 재매수

> 추론 보충 범주: "거시 리스크 내성 판단" — 금리 충격에 대한 한국 시장의 내성이 과거보다 높아졌다는 시장 컨센서스

## GPS 구조화

### 그룹 구조 (분기형 Branch)

```
한국 주식 시장 투자 판단
├── 금리 노이즈는 관리 가능하다
│   ├── BOJ 금리 1%대 — 예상 범위, 충격 제한
│   ├── 엔캐리 청산 우려 약화 (미-일 금리차 유지)
│   └── FOMC 동결 + 도비시 → 단기 변동성 축소
├── 반도체 슈퍼사이클은 아직 끝나지 않았다
│   ├── AI 하이퍼스케일러 투자 지속
│   ├── 고점 신호 미관찰
│   └── 삼성·하이닉스 이익 비중 70%
├── 반도체 주가 재평가는 멀티플 확장에 달려 있다
│   ├── 이익 성장 둔화
│   ├── M&A·주주환원 필요
│   └── 외국인 자금 유입 조건
└── 섹터 확산은 보조 전략
    ├── 반도체 공백기 한정
    └── 순환매 형태로 접근
```

### 비교 분석 (패러렐 — 금리 이슈별 충격 비교)

| 이슈             | 단기 충격 | 구조적 위협 | 현재 투자 행동     |
|------------------|-----------|-------------|--------------------|
| BOJ 기준금리 1%대 | 낮음      | 제한적      | 무시 가능          |
| 엔캐리 청산      | 낮음      | 잠재적      | 모니터링           |
| FOMC 동결        | 없음      | 없음        | 변동성 축소 기회   |
| 장기금리 구조적 상방 | 없음   | 높음        | 채권 선행 지표 주시 |

### 순서 분석 (시리즈)

```
거시 노이즈 판단 (금리 이슈 소화)
    → 사이클 지속성 확인 (슈퍼사이클 연장 여부)
        → 종목 재평가 조건 검토 (멀티플 확장 가능성)
            → 포트폴리오 구성 결론 (반도체 50% + 순환매)
                ↑
          [누락 탐지: 매도 타이밍 기준 — 유증/IPO 신호 발생 시 즉시 대응]
```

## 핵심 주장

> 금리 노이즈와 지정학 불확실성은 단기 충격에 그치므로, 반도체 슈퍼사이클이 유효한 지금은 삼성전자·SK하이닉스 중심 50% 이상 비중을 유지하되 멀티플 재평가 조건과 섹터 순환매를 병행하는 전략이 가장 현실적이다.

## 피라미드 구조

### 근거 1: 금리 노이즈는 단기 충격에 그치며 파국 시나리오 가능성은 낮다
- BOJ 금리 1%대 진입은 예상 범위 내 — 엔캐리 청산 우려 약화
- FOMC 기준금리 동결 + 도비시 뉘앙스 예상 → 단기 변동성 축소
- 장기금리 구조적 상방 압력은 실재하나 채권이 주식보다 먼저 흔들릴 것 (아직 아님)

### 근거 2: 반도체 슈퍼사이클은 AI 투자 지속으로 아직 연장 국면에 있다
- AI 하이퍼스케일러의 대규모 투자 지속 → 메모리·AI칩 수요 구조적 확대
- 고점 신호(유증/IPO 급증, 리파이낸싱 이슈)는 현재 관찰되지 않음
- 삼성전자·SK하이닉스가 코스피 이익의 약 70% 차지 → 시장 중심축 불변

### 근거 3: 반도체 주가 추가 상승은 이익 성장보다 멀티플 재평가에 달려 있다
- 가격 급등기 종료 → 이익 성장 기울기 완만화 예상
- 재평가 조건: M&A(보스턴 다이나믹스 등 Physical AI), 자사주 매입·주주환원 강화
- 외국인 장기 자금 유입 위해 ROE 개선과 경영진 비전 제시 필요

### 근거 4: 섹터 확산은 반도체 공백기에만 유효한 보조 전략이다
- 반도체 2종목이 횡보하는 구간에만 타 섹터 확산 — 독립 주도는 어려움
- 유효 섹터: 현대차(로보틱스/Physical AI), 삼성SDS(AI 최종단), 만도, 조선·원전, CS윈드
- 전략: 기판→장비→소재 순환매 형태로 접근

## MECE 검증

- 결과: **통과**
- 로직트리 규칙: 엄격 1~2단계 적용 / 3단계 이후 완화
- 검토 사항:
  - 근거 1(금리)과 근거 2(사이클)는 명확히 분리됨
  - 근거 3(멀티플 재평가)은 실행 전략 레이어로 독립적
  - 근거 4(섹터 확산)는 보조 전략으로 명확히 구분

## 요약

금리 관련 노이즈(BOJ·FOMC·장기금리)는 3인 공통으로 단기 충격에 그친다고 판단하며, 반도체 슈퍼사이클은 AI 하이퍼스케일러 투자 지속으로 연장 국면이다. 따라서 삼성전자·SK하이닉스 50% 이상 비중을 핵심으로 유지하되, 이익 성장 둔화에 대응한 멀티플 재평가(M&A·주주환원)와 공백기 섹터 순환매를 병행하는 전략이 현 시점에서 가장 현실적이다.
```

## 예시 3: 역방향 구조화 (Reverse Structuring)

**입력 텍스트**:
```
결론적으로 정보관리기술사 자격증을 따는 게 가장 좋다. 
SI/SM 10년 차에 Azure 경험을 가지고 있는데, 그냥 데이터 엔지니어로 가는 것보다 
자격증을 통해 관리·컨설팅 쪽으로 가는 게 더 안정적일 것 같다.
```

**주요 판단 포인트**:
- 입력이 결론으로 시작 → Reverse 경로 판단
- `identifiedConclusion`: "정보관리기술사 자격증을 따는 게 가장 좋다"
- `retroQuestions`: 왜 더 안정적인가? Azure 경험은 어떻게 활용되는가?
- 이후 정방향 피라미드로 재구성

**생성된 사용자 출력 (Markdown)**:

```markdown
---
tags: [minto, think]

## CS 분석 (범주·요점)

### 범주 1: AI가 대체할 수 있는 영역
- 요점: 반복적 작업, 패턴 인식, 정보 요약, 단순 코딩

### 범구 2: AI가 대체하기 어려운 영역
- 요점: 모호한 문제 정의, 이해관계 조정, 책임지는 판단, 새로운 맥락 창출

> 추론 보충 범주: "경력 전략" — "대체당하는 사람" vs "AI를 사용하는 사람"의 분기

## GPS 구조화

### 그룹 구조

```
AI 시대 직업 생존
├── AI가 잘하는 것 (Contain)
│   ├── 반복 작업
│   ├── 데이터 처리
│   └── 표준화된 산출물
└── 인간이 더 잘하는 것 (Branch)
    ├── 문제 정의
    ├── 인간 관계
    └── 책임과 판단
```

## 핵심 주장

> AI는 특정 업무를 대체하지만, 문제 정의·판단·관계 관리 역량을 가진 사람은 오히려 더 가치가 높아진다.

## 피라미드 구조

### 근거 1: AI는 이미 많은 지식 노동의 하위 작업을 빠르게 대체하고 있다
- 코딩, 문서 작성, 데이터 분석의 상당 부분이 자동화됨

### 근거 2: 진짜 가치 있는 일은 "무엇을 할 것인가"를 정의하는 일이다
- AI는 "어떻게"는 잘하지만 "왜", "무엇을"은 아직 약함

### 근거 3: 인간만이 할 수 있는 최종 책임과 신뢰 형성이 경쟁력이 된다
- 고객과의 관계, 팀 내 신뢰, 리스크를 감수하는 판단

## MECE 검증

- 결과: 통과
- 입력이 매우 짧아 가정이 많음. "대체"와 "보완"을 명확히 구분하여 MECE 확보

## 요약

AI 때문에 직업이 없어질까 두려워하는 것은 자연스럽지만, 핵심은 "AI가 대체하는 일"과 "인간만이 할 수 있는 일"을 구분하는 데 있다. 후자에 집중하는 사람이 장기적으로 살아남는다.

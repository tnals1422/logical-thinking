## 패턴 가이드 (에이전트 참고 — 출력에 포함하지 않음)

**이 템플릿 사용 원칙**
- SCQA는 Minto 기본형. 대부분의 일반 보고서, 분석, 제안에 우선 적용.
- 핵심은 "상황 → 문제로 인한 긴장 → 그에 대한 질문 → 답변" 흐름.
- Answer 섹션은 피라미드 구조 (coreClaim + 3~5개 Level-1 근거)를 최대한 반영.
- 개조식 + 표 + 다이어그램 우선. 긴 narrative는 최소화.
